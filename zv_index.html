¿Qué hace la app?

Dos módulos. Una sola solución.

Gestión Financiera

Control de ingresos, gastos y metas de ahorro con alertas personalizadas y educación gamificada.

  • Presupuesto mensual inteligente
  • Metas de ahorro con progreso visual
  • Alertas antes de gastar de más
  • Lecciones financieras breves

Apoyo Emocional IA

Acompañamiento conversacional disponible 24/7, detecta estrés y ofrece técnicas de bienestar.

  • Chat empático en español
  • Diario emocional con análisis
  • Técnicas de manejo del estrés
  • Recursos universitarios locales

Stack Tecnológico

Construida con herramientas modernas para funcionar en Android e iOS con rendimiento óptimo.

  • Flutter + Dart (multiplataforma)
  • Firebase Firestore en tiempo real
  • API de NLP (OpenAI / Gemini)
  • GitHub para control de versiones
